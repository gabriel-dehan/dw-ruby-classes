--- title: Backup cc reveal: theme: solarized slideNumber: true progress: true mouseWheel: true transition: linear --- --- ## GEMS Ruby possède un outil permettant d'installer et travailler facilement avec des bibliothèques externes (libraries): les gems. -- ### Trouver une Gem - [https://rubygems.org/](https://rubygems.org/) - [https://www.ruby-toolbox.com/](https://www.ruby-toolbox.com/) - Google/Github -- ### Installer une Gem ```bash gem install nom_de_la_gem ``` Prenons l'exemple de [colorize](https://github.com/fazibear/colorize) ```bash $ gem install colorize ``` -- ### Utiliser une Gem ```ruby # mon_fichier.rb require 'nom_de_la_gem' ``` Donc pour colorize: ```ruby # colors.rb require 'colorize' colors = String.colors.map do |color_name| color_name.to_s.colorize(color_name) end puts colors.join(", ") puts "--------------" puts "Please my eyes".green.on_red ``` {% img 'example.png' alt:'Example gem' class:'example-gem' %}
